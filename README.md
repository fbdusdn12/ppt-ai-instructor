# PPT 기반 강의 설명 AI 강사 제작

## 📌 프로젝트 개요

PPT 파일을 입력하면 슬라이드 내용을 분석하고, 이를 바탕으로 강의하듯 설명해주는 AI 강사를 제작한 프로젝트입니다.

PPT의 텍스트, 이미지, 표 등 다양한 정보를 활용하여 슬라이드별 설명문을 생성하고, 자연스러운 강의 흐름으로 구성하는 것을 목표로 했습니다.

---

## 🎯 프로젝트 목표

- PPT 파일의 슬라이드 내용을 분석
- 슬라이드별 핵심 내용을 바탕으로 설명문 생성
- 강의자처럼 자연스럽게 설명하는 AI 서비스 구현
- 입력부터 결과 생성까지 이어지는 전체 워크플로우 구성
- 팀원들이 작성한 기능 코드를 하나의 실행 가능한 코드로 통합

---

## 🙋‍♂️ 나의 역할

**워크플로우 코드 통합 및 최종 실행 검수 담당**

팀원들이 작성한 PPT 분석, 설명문 생성, 검색, 음성/영상 생성 관련 코드를 받아 하나의 실행 흐름으로 통합했습니다.  
각 단계에서 생성된 결과가 다음 단계로 정상적으로 전달되는지 확인하고, 파일 경로, 상태값, 출력 결과와 관련된 오류를 수정했습니다.

## ✅ 결과 및 배운 점

이 프로젝트를 통해 AI 서비스는 개별 기능 구현뿐만 아니라, 입력부터 결과 생성까지 이어지는 전체 파이프라인 설계가 중요하다는 것을 배웠습니다.

특히 PPT 분석, 설명문 생성, 음성/영상 생성처럼 여러 단계가 연결된 구조에서는 상태 관리와 파일 경로 관리가 전체 실행 안정성에 큰 영향을 준다는 점을 경험했습니다.

또한 팀원들의 코드를 통합하고 최종 실행 가능한 코드로 정리하는 과정에서 워크플로우 설계와 디버깅 역량을 기를 수 있었습니다.
